I think interactive visualizations are not only an effective way to explore data and communicate ideas with collaborators, but as a tool for sketching out or thinking about new statistical ideas.

Michael Bostock's amazing D3.js JavaScript library and comprehensive documentation has made it much easier to create these visualizations.
